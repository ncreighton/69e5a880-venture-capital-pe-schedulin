# Venture Capital & PE Scheduling and Booking Telegram Bot

This Telegram bot is designed specifically for venture capital (VC) and private equity (PE) firms to streamline scheduling and booking of meetings, calls, and events. Integrated directly into Telegram, it allows partners, analysts, and associates to manage their calendars without switching apps. The bot syncs with major calendar services (Google Calendar, Outlook), handles time zone conversions, and sends automated reminders. It supports multi-user access, so team members can view availability, 

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
